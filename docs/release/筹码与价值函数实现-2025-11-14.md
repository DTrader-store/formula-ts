## 问题描述

实现 6 个筹码分布和价值追踪函数，用于分析成本分布、获利盘计算、条件触发值提取等高级技术分析功能。这些函数是量化交易系统中不可或缺的工具。

### 需要实现的函数

1. **WINNER(close, volume, price)** - 价格位置的获利盘比例
2. **LWINNER(close, volume, price)** - 浮动获利盘比例（短期）
3. **COST(close, volume, percent)** - 获利比例对应的成本价
4. **VALUEWHEN(cond, X)** - 条件满足时 X 的值
5. **TOPRANGE(X, period)** - X 是否为近期最高
6. **LOWRANGE(X, period)** - X 是否为近期最低

#### 核心算法实现

**1. WINNER - 获利盘比例计算**

```typescript
export function WINNER(
  close: number[],
  volume: number[],
  targetPrice: number[],
  lookback: number = 100
): number[] {
  const result: number[] = new Array(close.length).fill(0);

  for (let i = 0; i < close.length; i++) {
    const startIdx = Math.max(0, i - lookback + 1);
    let totalVolume = 0;
    let winVolume = 0;

    // 计算成交量加权的筹码分布
    for (let j = startIdx; j <= i; j++) {
      totalVolume += volume[j];
      // 在目标价格以下买入的筹码即为获利筹码
      if (close[j] < targetPrice[i]) {
        winVolume += volume[j];
      }
    }

    result[i] = totalVolume > 0 ? winVolume / totalVolume : 0;
  }

  return result;
}
```

**算法说明**：
- 回溯指定周期内的成交数据
- 计算在目标价格以下买入的成交量占比
- 返回 0-1 范围的获利盘比例

**2. COST - 成本价计算**

```typescript
export function COST(
  close: number[],
  volume: number[],
  percent: number[],
  lookback: number = 100
): number[] {
  const result: number[] = new Array(close.length).fill(0);

  for (let i = 0; i < close.length; i++) {
    const startIdx = Math.max(0, i - lookback + 1);

    // 构建价格-成交量对
    const priceVolumePairs: Array<{ price: number; volume: number }> = [];
    let totalVolume = 0;

    for (let j = startIdx; j <= i; j++) {
      priceVolumePairs.push({ price: close[j], volume: volume[j] });
      totalVolume += volume[j];
    }

    // 按价格升序排序
    priceVolumePairs.sort((a, b) => a.price - b.price);

    // 找到累积成交量达到目标百分比的价格
    const targetVolume = (totalVolume * percent[i]) / 100;
    let cumulativeVolume = 0;

    for (const pair of priceVolumePairs) {
      cumulativeVolume += pair.volume;
      if (cumulativeVolume >= targetVolume) {
        result[i] = pair.price;
        break;
      }
    }
  }

  return result;
}
```

**算法说明**：
- 对回溯期内的价格按升序排序
- 计算累积成交量分布
- 返回达到指定获利比例的成本价

**3. VALUEWHEN - 条件触发值捕获**

```typescript
export function VALUEWHEN(cond: number[], X: number[]): number[] {
  const result: number[] = new Array(cond.length).fill(0);
  let lastValue = 0;

  for (let i = 0; i < cond.length; i++) {
    // 条件满足时捕获值
    if (cond[i] !== 0) {
      lastValue = X[Math.min(i, X.length - 1)];
    }
    result[i] = lastValue;
  }

  return result;
}
```

**算法说明**：
- 当条件满足（非零）时，捕获 X 的当前值
- 保持该值直到条件再次满足
- 常用于记录买入/卖出信号时的价格

**4. TOPRANGE/LOWRANGE - 极值判断**

```typescript
export function TOPRANGE(X: number[], period: number = 20): number[] {
  const lookback = Math.floor(period);
  const result: number[] = new Array(X.length).fill(0);

  for (let i = 0; i < X.length; i++) {
    const startIdx = Math.max(0, i - lookback + 1);

    // 找到范围内的最大值
    let maxValue = -Infinity;
    for (let j = startIdx; j <= i; j++) {
      if (X[j] > maxValue) {
        maxValue = X[j];
      }
    }

    // 检查当前值是否等于最大值
    result[i] = X[i] === maxValue ? 1 : 0;
  }

  return result;
}
```


### 5. 组合策略

```typescript
// 综合筹码分析策略
// 1. 获利盘较少（套牢盘多）

// 2. 价格接近成本密集区

// 综合买入信号
BUY: LOW_WINNER AND NEAR_COST AND BREAKOUT;
```

## 细节与备注

### 算法简化

1. **筹码分布算法**
   - 使用成交量加权的简化算法
   - 不考虑流通股本的稀释和增发
   - 假设所有成交都是换手（简化模型）
   - 适用于趋势分析，不适用于精确持仓分析

2. **性能优化**
   - 默认回溯周期限制（WINNER: 100, LWINNER: 20）
   - 使用滑动窗口避免全量计算
   - 未实现增量计算缓存（可作为未来优化方向）

### 边界处理

1. **数据不足**
   - 前 N 根 K 线数据不足时，使用可用数据计算
   - 不返回 NaN，保证结果数组完整性

2. **特殊值处理**
   - 零成交量时返回 0（避免除零错误）
   - 空数组返回空数组
   - 小数周期参数自动向下取整

3. **数组长度不一致**
   - VALUEWHEN 支持不同长度的 cond 和 X 数组
   - 使用 Math.min 确保不越界
