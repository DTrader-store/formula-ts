# 完整演示页面实现

**日期**: 2025-11-14

## 问题描述

需要创建一个完整的演示页面，展示 Formula-TS 公式解析引擎的功能。要求包括：
- 公式编辑器，支持实时编辑和语法检查
- 集成 TradingView Lightweight Charts 显示 K 线图和技术指标
- 提供多个预设的公式示例（MA、MACD、KDJ、BOLL、RSI 等）
- 现代化的深色主题 UI
- 响应式设计，支持桌面和移动设备
- 完整的交互功能：加载数据、执行公式、查看结果

## 分析过程

1. **技术栈选择**:
   - 使用 TradingView Lightweight Charts 作为图表库（轻量级、高性能）
   - Vite 作为开发服务器和构建工具（快速、零配置）
   - 纯 TypeScript + Vanilla JS（无额外框架依赖）

2. **架构设计**:
   - 分离关注点：HTML 结构、CSS 样式、TypeScript 逻辑
   - 示例数据和公式单独管理（examples.ts）
   - 使用安全的 DOM 操作避免 XSS 漏洞

3. **功能模块**:
   - 图表初始化和管理（主图表 + 指标图表）
   - 公式执行引擎集成
   - 事件处理（按钮点击、示例选择等）
   - 结果展示（图表渲染 + 文本输出）

### 5. 主应用逻辑 (`main.ts`)

#### 核心功能实现：

**图表初始化**:
```typescript
function initCharts(): void {
  // 创建主图表（K线和叠加指标）
  mainChart = createChart(mainChartContainer, chartOptions);

  // 创建指标图表（独立指标如 MACD、RSI）
  indicatorChart = createChart(indicatorChartContainer, chartOptions);

  // 创建 K 线序列
  candlestickSeries = mainChart.addCandlestickSeries({...});

  // 同步时间轴
  mainChart.timeScale().subscribeVisibleTimeRangeChange(...);
}
```

**公式执行**:
```typescript
function executeFormula(): void {
  const formula = editor.value.trim();
  const result = formulaEngine.evaluate(formula, marketData);

  displayResults(result);
  drawIndicators(result);
}
```

## 功能演示

### 如何运行

1. **安装依赖**:
   ```bash
   npm install
   ```

2. **启动演示**:
   ```bash
   npm run demo
   ```

   页面将在浏览器中自动打开: `http://localhost:3000/demo/index.html`

3. **构建生产版本**:
   ```bash
   npm run demo:build
   ```

### 使用流程

1. 点击 "加载示例数据" 加载 100 条 K 线数据
2. 从下拉菜单选择示例公式（如 "移动平均线 (MA)"）
3. 点击 "执行公式" 按钮
4. 在图表上查看技术指标叠加效果
5. 在结果面板查看具体数值

### 支持的公式语法

```
// 变量声明
MA5: MA(CLOSE, 5);
MA10: MA(CLOSE, 10);

// 输出绘制
DRAW MA5, COLOR: RED;
DRAW MA10, COLOR: BLUE;

// 条件判断
SIGNAL: IF(CROSS(MA5, MA10), 100, 0);

// 复合计算
DIF: EMA(CLOSE, 12) - EMA(CLOSE, 26);
DEA: EMA(DIF, 9);
```

2. **时间轴**:

3. **错误处理**:
   - 错误消息可以更详细（行号、列号）
