# Phase 2 - Task 3: 扩展逻辑函数

## 日期
2025-11-14

## 问题描述
需要在 `logical.ts` 中实现四个新的逻辑函数，以扩展公式引擎的逻辑运算能力：
- **EVERY(X, N)**: 检查N周期内X是否都满足（都非零）
- **EXIST(X, N)**: 检查N周期内是否存在X满足（至少一个非零）
- **BARSLAST(X)**: 计算上次X非零到现在的周期数
- **COUNT(X, N)**: 统计N周期内X非零的次数

## 分析过程
1. 首先审查现有的逻辑函数实现（IF 和 CROSS），了解代码风格和约定
2. 分析了四个新函数的功能规格和预期行为
3. 按照 TDD 流程，先编写完整的测试用例
4. 识别到测试用例中的一个逻辑问题（EVERY函数在N=2的测试中），并及时修正
5. 逐个实现四个函数，确保与测试用例完全匹配

## 解决方案

### 1. 实现 EVERY 函数
检查最后N个周期内的所有值是否都非零：
- 前N-1个值返回 NaN（数据不足）
- 从第N个值开始，检查之前N个值是否都非零
- 返回 1（全非零）或 0（至少一个零）

示例：EVERY([1,1,0,1], 3) → [NaN, NaN, 0, 0]

### 2. 实现 EXIST 函数
检查最后N个周期内是否至少存在一个非零值：
- 前N-1个值返回 NaN（数据不足）
- 从第N个值开始，检查之前N个值中是否存在非零
- 返回 1（存在非零）或 0（全零）

示例：EXIST([0,0,1,0], 3) → [NaN, NaN, 1, 1]

### 3. 实现 BARSLAST 函数
计算距离上一个非零值的周期数：
- 当前值非零时，返回 0
- 当前值为零但存在之前的非零值时，返回距离值
- 没有之前的非零值时，从 1 开始递增

示例：BARSLAST([0,1,0,0,1]) → [1, 0, 1, 2, 0]

### 4. 实现 COUNT 函数
统计最后N个周期内非零值的个数：
- 前N-1个值返回 NaN（数据不足）
- 从第N个值开始，计算之前N个值中的非零个数
- 返回计数结果（0 到 N）

示例：COUNT([1,0,1,1,0], 3) → [NaN, NaN, 2, 2, 2]

## 细节与备注

### 函数设计决策

1. **NaN 处理**：EVERY、EXIST 和 COUNT 在数据不足时返回 NaN，符合技术分析中的常见做法

2. **非零定义**：所有函数都使用 `!== 0` 来判断非零，包括正数、负数、小数

3. **BARSLAST 的特殊处理**：
   - 当前值非零时返回 0（刚好出现）
   - 没有历史非零值时从 1 开始计数（未来可能出现）
