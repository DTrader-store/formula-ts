### 1. 示例公式创建

#### MA (移动平均线) - `examples/formulas/ma.txt`
```
# Short-term MA (5 days)
MA5:MA(CLOSE,5);

# Medium-term MA (10 days)
MA10:MA(CLOSE,10);

# Long-term MA (20 days)
MA20:MA(CLOSE,20);

# Golden Cross Signal
GOLDEN_CROSS:CROSS(MA5,MA10);

# Death Cross Signal
DEATH_CROSS:CROSS(MA10,MA20);
```

#### MACD (平滑异同移动平均线) - `examples/formulas/macd.txt`
```
# Fast EMA (12 periods)
FAST:EMA(CLOSE,12);

# Slow EMA (26 periods)
SLOW:EMA(CLOSE,26);

# MACD Line
MACD:FAST-SLOW;

# Signal Line
SIGNAL:EMA(MACD,9);

# Histogram
HIST:MACD-SIGNAL;

# Buy/Sell Signals
BUY:CROSS(MACD,SIGNAL);
SELL:CROSS(SIGNAL,MACD);
```

#### KDJ (随机指标) - `examples/formulas/kdj.txt`
```
# Period definitions
N:9;
M1:3;
M2:3;

# Highest High and Lowest Low
HH:HHV(HIGH,N);
LL:LLV(LOW,N);

# RSV calculation
RSV:(CLOSE-LL)/(HH-LL)*100;

# K, D, J lines
K:EMA(RSV,M1);
D:EMA(K,M2);
J:3*K-2*D;

# Trading signals
OVERBOUGHT:IF(K>80,1,0);
OVERSOLD:IF(K<20,1,0);
GOLDEN:CROSS(K,D);
DEATH:CROSS(D,K);
```

### 2. 示例市场数据

创建了 `examples/data/sample-market-data.json`，包含：
- **50 个完整的 OHLCV 数据点**
- 价格从 100.00 逐步上涨到 131.50
- 成交量从 1,000,000 增长到 2,750,000
- 每个数据点都包含：
  - open: 开盘价
  - high: 最高价
  - low: 最低价
  - close: 收盘价
  - volume: 成交量
  - amount: 成交额

数据特点：
- 呈现上升趋势，适合测试移动平均线和趋势指标
- 包含波动，可以测试震荡指标（如 KDJ）
- 真实的价格关系（high >= low，close 在范围内）
