# 基础行情字段函数实现

**日期**: 2025-11-14  
**任务**: Task 1 - 实现8个基础行情字段访问函数

## 问题描述

需要实现 8 个基础行情字段访问函数，允许公式直接访问 MarketData 中的字段，包括：
- 必需字段：OPEN, HIGH, LOW, CLOSE, VOL
- 可选字段：AMOUNT, ADVANCE, DECLINE

这些函数是公式系统的基础，为后续的技术指标计算提供数据源。

## 分析过程

### 1. 架构分析

发现项目使用了特定的架构模式：
- 函数通过 `Interpreter.visitFunctionCall()` 的 switch 语句调用
- 市场数据字段原本通过 `Context.getMarketDataField()` 访问
- 需要将字段访问改为函数调用方式

### 2. 设计决策

**函数接口**:
```typescript
export interface MarketDataFunction {
  name: string;
  minArgs: number;    // 0 - 无参数函数
  maxArgs: number;    // 0 - 无参数函数
  execute: (args: number[][], context: ExecutionContext) => number[];
}
```

**可选字段处理策略**:
- 使用 `isNaN()` 检测缺失数据（Context 返回 NaN）
- 抛出友好的错误消息
- 对指数专用字段添加额外提示

### 3. 关键见解

- 所有现有函数都需要参数，这是第一批无参数函数
- 需要在解释器中正确处理空参数列表
- 可选字段的错误处理至关重要，避免用户困惑

## 解决方案

### 实现的函数

| 函数 | 描述 | 可选 | 错误处理 |
|------|------|------|----------|
| **OPEN()** | 开盘价 | 否 | - |
| **HIGH()** | 最高价 | 否 | - |
| **LOW()** | 最低价 | 否 | - |
| **CLOSE()** | 收盘价 | 否 | - |
| **VOL()** | 成交量 | 否 | - |
| **AMOUNT()** | 成交额 | 是 | 提示提供 amount 字段 |
| **ADVANCE()** | 涨家数 | 是 | 提示仅适用于指数数据 |
| **DECLINE()** | 跌家数 | 是 | 提示仅适用于指数数据 |

### 核心实现

**1. 基础函数示例**:
```typescript
export const CLOSE: MarketDataFunction = {
  name: 'CLOSE',
  minArgs: 0,
  maxArgs: 0,
  execute: (_args, context) => {
    return context.getMarketDataField('CLOSE');
  }
};
```

**2. 可选字段处理**:
```typescript
export const AMOUNT: MarketDataFunction = {
  name: 'AMOUNT',
  minArgs: 0,
  maxArgs: 0,
  execute: (_args, context) => {
    const result = context.getMarketDataField('AMOUNT');
    
    if (result.some(v => isNaN(v))) {
      throw new Error(
        'AMOUNT function requires marketData.amount field. ' +
        'Please provide the "amount" field in your market data.'
      );
    }
    
    return result;
  }
};
```

**3. 解释器集成**:
```typescript
// 在 Interpreter.visitFunctionCall() 中
case 'CLOSE':
  if (args.length !== 0) {
    throw new Error(`CLOSE expects 0 arguments, got ${args.length}`);
  }
  return marketDataFunctions.CLOSE.execute(args, this.context);
```
