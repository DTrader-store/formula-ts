# Phase 2 - Task 2: 统计函数实现

**日期**: 2025-11-14

## 问题描述

为公式解释器实现统计函数模块，包含以下四个核心统计函数：
- **STD(X, N)**: N周期标准差
- **VAR(X, N)**: N周期方差
- **MEDIAN(X, N)**: N周期中位数
- **AVEDEV(X, N)**: N周期平均绝对偏差

## 分析过程

### 1. 架构调研
通过查看现有的数学函数实现（MA、EMA、SUM等）和参考函数（REF、HHV、LLV），确定了统计函数的设计模式：
- 所有函数接收数组和周期参数
- 返回相同长度的结果数组，不足周期的位置返回NaN
- 使用人口统计（population statistics），不是样本统计

### 2. 测试驱动开发（TDD）流程
按照严格的TDD流程实现：
1. 先编写全面的测试套件
2. 然后实现功能
3. 最后验证所有测试通过

## 解决方案

### 1. 创建统计函数实现文件

**文件**: `/src/interpreter/functions/statistics.ts`

实现了四个统计函数：

#### STD - 标准差
```typescript
export function STD(data: number[], period: number): number[]
```
- 计算每个滑动窗口的人口标准差
- 公式: σ = √[Σ(x - μ)² / N]
- 示例: STD([1,2,3,4,5], 3) → [NaN, NaN, 0.8165, 0.8165, 0.8165]

#### VAR - 方差
```typescript
export function VAR(data: number[], period: number): number[]
```
- 计算每个滑动窗口的人口方差
- 公式: σ² = Σ(x - μ)² / N
- 示例: VAR([1,2,3,4,5], 3) → [NaN, NaN, 0.6667, 0.6667, 0.6667]

#### MEDIAN - 中位数
```typescript
export function MEDIAN(data: number[], period: number): number[]
```
- 计算每个滑动窗口的中位数
- 支持奇数窗口大小（返回中间值）和偶数窗口大小（返回两个中间值的平均值）
- 示例: MEDIAN([1,3,2,5,4], 3) → [NaN, NaN, 2, 3, 4]

#### AVEDEV - 平均绝对偏差
```typescript
export function AVEDEV(data: number[], period: number): number[]
```
- 计算每个滑动窗口的平均绝对偏差
- 公式: AAD = Σ|x - μ| / N
- 示例: AVEDEV([1,2,3,4,5], 3) → [NaN, NaN, 0.6667, 0.6667, 0.6667]

## 细节与备注

### 实现要点
1. **人口统计 vs 样本统计**: 选择使用人口统计（除以N），与其他函数保持一致
2. **NaN处理**: 遵循现有约定，不足周期的位置返回NaN
3. **边界情况**: 全面处理了period=1、period>length、相同值等边界情况

### 性能特性
- **时间复杂度**:
  - STD、VAR、AVEDEV: O(n*period) - 每个数据点都要计算统计量
  - MEDIAN: O(n*period*log(period)) - 包含排序操作
- **空间复杂度**: O(n) - 返回与输入同长的数组

### 函数交互
这些统计函数可以与其他函数组合使用，例如：
- `STD(MA(CLOSE, 5), 10)` - 计算5周期MA的10周期标准差
- `IF(AVEDEV(CLOSE, 5) > 2, 1, 0)` - 当平均绝对偏差超过2时触发

### 未来扩展
- 可添加样本标准差/方差的变体（使用N-1）
- 可添加加权统计函数
- 可考虑性能优化（使用Welford算法计算方差）
