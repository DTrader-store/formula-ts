# VSCode TDX 公式语法高亮扩展

## 日期
2025-11-14

## 问题描述

为了提升开发者体验，需要创建一个 VSCode 扩展，为 TDX（通达信）公式语言提供专业的语法高亮功能。这个扩展应该支持项目中定义的所有关键字、函数、操作符和特殊语法元素。

## 分析过程

### 1. 需求分析
- 分析项目中的 Lexer 和 TokenType 定义，确定所有关键字和操作符
- 梳理 FunctionRegistry 中注册的所有函数（24个）
- 确定市场数据字段（OPEN, CLOSE, HIGH, LOW, VOLUME, AMOUNT）
- 识别图表属性（颜色、线条厚度、线条样式）
- 理解注释规则（行注释 // 和块注释 {}）

### 2. 技术选型
- 使用 TextMate 语法定义（.tmLanguage.json）
- 创建语言配置文件支持编辑器功能
- 编写代码片段提升开发效率
- 提供完整的安装和使用指南

### 3. 文件结构设计
```
vscode-extension/
├── package.json                    # 扩展元数据
├── language-configuration.json     # 语言配置
├── syntaxes/
│   └── tdx-formula.tmLanguage.json # 语法定义
├── snippets/
│   └── snippets.json              # 代码片段
├── examples/
│   └── example.tdx                # 示例文件
├── README.md                       # 使用文档
└── INSTALLATION_GUIDE.md           # 安装指南
```

## 解决方案

### 1. 核心实现

#### 文件：package.json
- 定义扩展名称、版本、发布者
- 配置 VSCode 最低版本要求（1.50.0）
- 关联文件类型：.tdx 和 .formula
- 指定语言 ID 和配置文件路径
- 配置语法定义和代码片段

#### 文件：language-configuration.json
实现编辑器级别的功能：
- 注释支持：单行（//）和块注释（{}）
- 括号匹配：[] {} ()
- 自动闭合对：引号、括号
- 自动缩进规则
- 代码折叠标记

#### 文件：syntaxes/tdx-formula.tmLanguage.json
定义 TextMate 语法：
- **注释**：双斜杠和块注释
- **字符串**：单引号和双引号
- **数字**：整数和浮点数
- **操作符**：
  - 算术：+ - * /
  - 比较：> < >= <= = <>
  - 赋值：:=
  - 逻辑：AND OR
- **关键字**：IF THEN ELSE AND OR NOT
- **函数**：24 个内置函数
  - 数学：MA, EMA, SUM, MAX, MIN, ABS, SQRT, POW, MOD, ROUND
  - 引用：REF, HHV, LLV
  - 逻辑：CROSS, EVERY, EXIST, BARSLAST, COUNT
  - 统计：STD, VAR, MEDIAN, AVEDEV
  - 技术面：SMA, WMA, RSI, BOLL, ATR
- **市场数据**：OPEN, CLOSE, HIGH, LOW, VOLUME, AMOUNT
- **图表属性**：
  - 8 种颜色：COLORRED, COLORGREEN, COLORBLUE 等
  - 10 级线条厚度：LINETHICK0-9
  - 2 种线条样式：DOTLINE, STICK

#### 文件：snippets/snippets.json
20+ 代码片段覆盖：
- 函数调用：ma, ema, sum, max, min, ref, hhv, llv 等
- 条件语句：if, cross, every, exist, barslast, count
- 变量赋值：var
- 绘图命令：drawred, drawgreen, drawblue
- 注释：comment, bcomment

## 细节与备注

### 语法覆盖范围

#### 关键字（来自 Lexer.ts）
- IF
- AND
- OR
- DOTLINE（已在 TokenType 中）
- STICK（已在 TokenType 中）

#### 函数（来自 FunctionRegistry）
24 个内置函数：
- Math: MA, EMA, SUM, MAX, MIN, ABS, SQRT, POW, MOD, ROUND
- Reference: REF, HHV, LLV
- Logical: IF, CROSS, EVERY, EXIST, BARSLAST, COUNT
- Statistics: STD, VAR, MEDIAN, AVEDEV
- Technical: SMA, WMA, RSI, BOLL, ATR

#### 市场数据（来自 MarketData.ts）
- OPEN（开盘价）
- CLOSE（收盘价）
- HIGH（最高价）
- LOW（最低价）
- VOLUME（成交量）
- AMOUNT（成交额）

#### 操作符（来自 TokenType）
- 算术：PLUS(+), MINUS(-), MULTIPLY(*), DIVIDE(/)
- 比较：GT(>), LT(<), GTE(>=), LTE(<=), EQ(=), NEQ(<>)
- 赋值：ASSIGN(:=)
- 逻辑：AND, OR

#### 图表属性（来自 Lexer.ts）
- 颜色：COLOR* 类
- 线条厚度：LINETHICK* 类（LINETHICK0-9）
- 线条样式：DOTLINE, STICK

3. **从 VSCode 找到**：通过扩展菜单找到 extensions 文件夹
